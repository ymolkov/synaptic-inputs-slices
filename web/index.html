CLAMP turns stepped recordings into phase-resolved synaptic drive.

The method pools current and voltage samples by respiratory phase, fits local I-V relationships, estimates each recording's inhibitory reversal geometry, then separates excitation and inhibition as normalized conductances.

Problem

A respiratory neuron records one mixed current. The circuit asks for two drives.

Each whole-cell trace contains leak, excitation, inhibition, command steps, and respiratory-cycle timing. CLAMP makes the decomposition tractable by treating respiratory phase as the organizing coordinate: samples from many cycles and command levels are pooled into phase bins, so each moment of the rhythm gets its own current-voltage estimate.

Method Map

From command steps to interpretable conductance fingerprints.

The pipeline is organized around five transformations. Each transformation preserves the same phase clock, so the final traces and circuit diagram can be read back against the respiratory cycle.

Stage 01

Stepped acquisition supplies the I-V spread.

Current-clamp and voltage-clamp command levels are held piecewise constant while the slice continues to cycle. The analysis uses the whole accepted interval rather than a single step.

Input
Aligned current, voltage, and XII traces
Output
Usable command epochs across many cycles

Inference Geometry

The wedge is the method's hinge.

In each phase bin, CLAMP fits a robust line through the pooled I-V samples: I = GtotV + I0. Across the cycle, those fitted slopes and intercepts trace a cloud in (Gtot, I0) space.

The upper envelope of that cloud estimates the recording-specific inhibitory reversal boundary. With Ei, Ee, and the leak-only limits fixed, the geometry resolves the mixed synaptic current into Gexc/gleak and Ginh/gleak.

Circuit Readout

A reduced slice network with a strong inhibitory scaffold.

The reconstructed conductances support a compact preBotC circuit: a VgluT2 inspiratory rhythmogenic kernel, tonic expiratory inhibition from VGAT-E neurons, and phasic inspiratory inhibition from VGAT-I neurons.

  • Excitation is concentrated in the inspiratory phase and supports the VgluT2-I kernel.
  • Inhibition appears as tonic expiratory drive and phasic inspiratory drive across populations.
  • VgluT2-E is rare in the slice data and appears largely passive in the active expiratory interval.

Interpretation

The slice keeps the rhythmogenic core and simplifies the intact connectome.

The companion manuscript frames the slice as a functionally reduced state. Instead of the richer three-phase in situ sequence, the isolated preparation preserves a compact excitatory kernel plus an inhibitory half-center motif that organizes inspiratory-expiratory switching.
